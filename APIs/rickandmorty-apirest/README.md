# Rick And Morty - API REST

## :gear: Setup
1. Install npm
```
npm install
```
2. Install Axios
```
npm install axios
```

3. Start the app
```
npm start
```

## :wrench: Built with

- [Axios](https://github.com/axios/axios)
- [Rick And Morty API REST](https://rickandmortyapi.com/documentation/#rest)
