# Weather - API REST

## :gear: Setup
1. Install npm
```
npm install
```
2. Install Axios
```
npm install axios
```

3. Start the app
```
npm start
```

## :wrench: Built with

- [Axios](https://github.com/axios/axios)
- [OpenWeatherMap API](https://openweathermap.org/api)
- [MapBox API](https://docs.mapbox.com/api/)
